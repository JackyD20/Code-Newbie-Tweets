I created this project as a way to practice querying an API, storing information from that API in a database, and creating a new API endpoint serving up information from my database. Along the way, I also wanted to practice front-end and design skills.

I used code newbie tweets for the project because I love the code newbie community! If you're interested in contributing to this project, please feel free to fork the repository and make some changes. You can look at existing issues in github for inspiration! If you'd like to add your github handle to the list of contributors, please add it to the friends page and submit that along with your changes your pull request!

This project is built using python and flask. In doing this for the first time, I owe thanks to @osaetindaniel for this excellent blog post. If you're eager to get started but not sure where to start you may find it useful to peruse the Flask documentation or this guide on how to submit a pull request.
